# NOI2025 省队名额划分规则

以下是NOI2025省队名额划分的详细规则：

## 一、A类名额（基本选手名额）
每个省份有5个A类名额，其中至少一个名额保留给女选手。

## 二、B类名额（优秀奖励选手名额）
B类名额是根据省份在NOI活动中的表现而设定的，由B1、B2、B3三部分组成。一个省份的总B类名额（S）是这三部分的总和。

1.  **B1（普及项）：** 占B类名额的50%，根据各省在最近一次全国青少年信息学奥林匹克联赛（NOIP）中的参赛人数占全国总参赛人数的比例计算。
    *   公式：B1 = S × 50% × (NOIP省级参赛人数 / 全国总参赛人数)。

2.  **B2（综合项）：** 占B类名额的30%。根据各省在最近一次NOIP中的分数从高到低排列，并分为K1个区间。每个区间的平均分作为该省的代表分数。然后，将所有省份的代表分数进行全国排名，选取前S × 30%（向下取整）的代表分数。一个省份的B2名额由其被选中的代表分数数量决定。如果最后一个被选中的位置有并列分数，则所有并列分数都将被包含，并按比例调整名额。

3.  **B3（拔尖项）：** 占B类名额的20%。根据各省在最近一次NOIP中的前K2个最高分作为优秀分数。然后，将所有省份的优秀分数进行全国排名，选取前S × 20%（向下取整）的优秀分数。一个省份的B3名额由其被选中的优秀分数数量决定。与B2类似，如果最后一个被选中的位置有并列分数，则所有并列分数都将被包含，并按比例调整名额。

### B类名额的附加规定：
*   一个省份的总B类名额与最近一次NOIP中非零分参赛人数的总和之比不得超过指定百分比P（默认为5%），且一个省份的总B类名额不得超过12个。
*   参数S、K1、K2和P将在执行时确定并公布。
*   B类名额将四舍五入到最接近的整数。
*   只有在NOIP中取得非零分的高中生才会被纳入这些计算。

## 三、省内选拔
各省负责NOI2025参赛选手的选拔过程。NOIP2024的成绩必须占最终省队选拔成绩的至少30%。省队选拔方案必须经省竞赛与技术委员会批准，并报送CCF审查。

---
**示例：**
NOI2025年广东省和浙江省各有17个名额（5个A类和12个B类），而甘肃、宁夏、青海等省份有5个名额（全部为A类）。
